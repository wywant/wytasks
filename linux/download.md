# curl
  * -L, --location      Follow redirects (H) 
  * -s, --silent        Silent mode (don't output anything)
  
## curl -sL https://deb.nodesource.com/setup_8.x
这是nodejs安装文档里面的命令， 支持重定向，并且不输出详细日志
