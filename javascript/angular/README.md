#Angular 2学习笔记

##简介
我学习的是TypeScript的例子

##文件结构

* index.html import了main.js
* main.ts import了app.module, 并且设置AppComponent为Bootstrap
* app.module import了app.component，hero-detail.component
* app.module设置了filter和template

* @Input
* @Output
